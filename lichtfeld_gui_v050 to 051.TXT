2. --init file type hint is incomplete (line 342)
The GUI description says .ply, .sog, .spz, .resume but V051 also accepts .usd, .usda, .usdc, .usdz. Minor cosmetic fix only — functionally fine if users manually type those extensions.
3. View file browser missing USD formats (lines 572–586)
browse_view_file() lists splat filetypes as *.ply *.sog *.spz but V051 now supports *.usd *.usda *.usdc *.usdz. The browse dialog won't show those files unless updated.
